启动程序：运行Timer_V1.0.py文件

关于这个压缩包内文件的说明：
1. record.txt 是一个我已经随机生成好的记录了时间的文件，以便向第一次使用程序的人展示日历效果。如果想要清空记录，可以直接点开该文件，删除所有的内容，保存。
2. rand_record.py 是一个用来随机生成时间记录的程序，它和Timer Lite主程序并不相关。你可以试着使用它生成一些随机记录，也可以修改它，让它按照你想要的方式生成记录。
3. .png和.TTF文件请不要删改，他们是程序的资源文件。
4. 如果有更多问题，请打开Timer Lite文档.pdf查看。本README文件只是用于说明一些打包进来了，但和程序本身无关的内容。
